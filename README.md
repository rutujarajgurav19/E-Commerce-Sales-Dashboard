# E-Commerce Sales Analytics Dashboard

## Project Overview

This project analyzes e-commerce sales performance using Power BI. The dashboard provides insights into sales trends, profitability, customer behavior, and shipping operations to support data-driven business decisions.

## Tools Used

* Power BI
* DAX
* Power Query
* Excel

## Key KPIs

* Total Sales: 2.30M
* Total Profit: 286.40K
* Total Quantity: 38K
* Total Discount: 358.83K
* Average Profit: 28.66

## Dashboard Pages

### 1. Overview Dashboard

Provides a summary of key business metrics and performance trends.

### 2. Sales & Revenue Analysis

Analyzes revenue trends across regions, products, and categories.

### 3. Profitability Analysis

Identifies high-profit and low-profit products, categories, and regions.

### 4. Customer Analysis

Examines customer segments and purchasing behavior.

### 5. Shipping & Operational Analysis

Evaluates shipping performance and operational efficiency.
